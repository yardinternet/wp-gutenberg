## Hooks

### yard-gutenberg/allowedBlocks

By default, all blocks will be registered. Use this filter to register only the allowed blocks.

```PHP
add_filter('yard::gutenberg/allowed-blocks', fn () => [
 'collapse',
 'collapse-item',
 'counting-number',
 'facetwp',
 'icon',
 'iconlist',
 'iconlist-item',
 'slide',
 'slider',
 'tabs',
 'tabs-item',
]);
```
